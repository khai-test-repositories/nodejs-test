
# nodejs-test
Quick Test repo for NodeJS and V8

## Prerequisites

 * NodeJS + npm
 * node-gyp
 * A proper C++ compiler
 * Git

## Get the code

```bash
git clone https://github.com/khai-test-repositories/nodejs-test.git
cd nodejs-test
```

## Building

```bash
npm run configure
npm run build
```

## Testing

```bash
npm test
```
