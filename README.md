
# nodejs-test
Quick Test repo for NodeJS and V8

## Prerequisites

 * NodeJS + npm
 * A proper C++ compiler
 * Git

## Get the code

```bash
git clone https://github.com/khai-test-repositories/nodejs-test.git
cd nodejs-test
```

## Initilization

```bash
npm install
npm run configure
```

## Building

```bash
npm run build
```

## Testing

```bash
npm test
```
